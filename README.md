# React Full-Stack

Just an exploration of building a React/Express app using newer tools/libraries/APIs.

## Usage

1. Create a `colors` database.
2. Run the queries from `/db/schema.sql`.
3. Run the queries from `/db/seed.sql`.
4. `npm install`.
5. `npm run server`.
6. `npm run client`.
7. Enjoy using this very advanced application.
